# Лог деплоев Apps Script (Дашборд_Apps_Script)

Одна строка на версию. Цель - чтобы любая сессия перед деплоем видела, что уже
задеплоено и кем, и не откатывала чужую работу вслепую.

**Правило перед КАЖДЫМ `clasp push`:** `clasp pull` -> `diff --strip-trailing-cr` с
`scripts/full_script_final.js` в git HEAD этого репозитория. Если diff показывает
что-то, чего нет в последнем коммите репозитория - это чужая недокоммiченная работа,
её нужно сначала сохранить (sync-коммит), и только потом вносить свои правки. После
`clasp push` + `create-version` + `redeploy` - сразу добавить строку в эту таблицу
(лог начат 2026-07-15, более ранние версии не восстановлены задним числом).
